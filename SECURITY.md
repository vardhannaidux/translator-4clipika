# Security Policy

## Supported Versions
Only the latest release (v1.0.0+) receives security patches.

## Reporting a Vulnerability
If you discover a security vulnerability, please do NOT open a public GitHub issue. Instead, report it privately by opening a ticket or emailing the security contact (security@telugudigitization.org) with replication details.
We aim to respond within 48 hours and patch vulnerabilities within 14 days.
