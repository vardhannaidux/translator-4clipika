# Contributing Guidelines

Thank you for your interest in contributing to the Eenadu 4C Lipika Translator Suite!

## Code of Conduct
Please be respectful and professional in all interactions.

## How to Contribute
1. Fork the repository on GitHub.
2. Create a feature branch: `git checkout -b feature/your-feature-name`.
3. Set up local development environment (refer to `README.md` for steps).
4. Run testing commands before opening pull requests:
   ```bash
   pytest
   ```
5. Commit clean, styled code following PEP 8 recommendations.
6. Open a Pull Request detailing the changes, reasoning, and test results.
